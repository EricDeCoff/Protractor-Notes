## Protractor ( Install )
```bash
*** Admin Command Prompt ( Windows ) ***
[sudo] npm install -g protractor
```

## Protactor ( Validate )
```bash
protractor --version
```

## WebDriver-Manager ( install )
```bash
webdriver-manager update
```

## WebDriver-Manager ( start )
```bash
webdriver-manager start
```
